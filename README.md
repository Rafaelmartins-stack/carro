# Meu Jogo Carro (Antigravity Edition)

Um jogo desenvolvido em Python com Pygame. Nele, você dirige um carro e desvia dos inimigos, mas com um diferencial incrível: o modo Antigravidade (o carro pode pular por cima dos obstáculos)!

## Como Jogar
- **Setas Direcionais:** Movimentam o carro (Cima, Baixo, Esquerda, Direita).
- **Barra de Espaço:** Ativa o salto antigravidade (permite sobrevoar os inimigos e não ser atingido).
- **Enter:** Reinicia a partida ao bater.

## Como Executar
Você precisa ter o Python instalado. É recomendado usar um ambiente virtual para não misturar as dependências do seu computador.

1. Instale o Pygame:
```bash
pip install -r requirements.txt
```

2. Execute o jogo:
```bash
python main.py
```
